Software as a service demonstrated by a mortgage calculator

My Computing Science BSc final year projects aim was to develop a fully functional and developer friendly software as a service (SAAS) system.

The core application was a mortgage calculator able to calculate mortgages for customers and suggest banks and mortgage products using an algorithm to match potential customers with banks. This application is then made universally accessible through the creation of a cloud hosted REST API.

Developers would then be able to build applications, websites or plug-ins that would use the mortgage calculator API to perform the mathematics whilst they have total control over how the resulting data is presented to the end-user.

As a demonstration of how the service could be utilized I have developed a web application and an android application which both utilize the same API yet contain distinct functionality and display the data returned in visually different ways.

I used Windows Azure to host the application in the cloud, creating a cloud service which provided the access to the applications API from any client connected to the World Wide Web. Developers would be able to view the documentation for the API which describes the functionality available through the service, with detailed examples of the request/response format.
